## Schnellübersicht

**Tech-Stack:**
- React 18 + TypeScript
- Vite (Build-Tool)
- Material-UI + Tailwind CSS
- Redux Toolkit + RTK Query
- Matterport SDK 1.4.24
- Clerk (Authentifizierung)

**Repositories:**
- Frontend: [cosmic-webapp-frontend](https://github.com/Graswald-Schaetzle/cosmic-webapp-frontend)
